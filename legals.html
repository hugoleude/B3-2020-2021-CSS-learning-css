Processus technique

Dans cette partie, je vais tâcher d'expliquer les propriétés que j'ai utilisé pour réaliser mon site.
Le site repose sur la propriété Grid CSS. Je ne connaissais pas bien cette propriété mais j'ai profité de ce cours pour
l'approfondir. Elle est notamment utilisée pour créer la grille présentant les personnalités. La grid contient
3 colonnes. Malheuresement je n'ai pas réussi à faire passer la grid de 3 à 1 colonne passant une certaine résolution. Le site
utilise aussi la propriété "Float". Le site fonctionne de manière responsive mise à part pour la présentation des personnalités.
Le footer, le header ainsi que les autres pages du site sont visibles entièrement.
